
SETUP PROYECT:

1. Ejecutar npm i en cada uno de los tres microservices
2. Crear una bd llamada transactions_db
3. Posicionarse en el microservice financial-transactions -> cd financial-transactions
4. En cd financial-transactions ejecutar -> npm run migration:run
5. Ejecutar microservicios posicionandose en su carpeta respectiva
	cd anti-fraud -> nest start anti-fraud
	cd financial-transactions -> nest start financial-transactions
	cd gql-gateway -> nest start gql-gateway

TEST PROYECT:

1. Para crear transaccion ejecutar el mutation:

mutation createFinancialTransaction(
    $tranferTypeId: Float!,
    $value: Float!
){
    createFinancialTransaction(
    tranferTypeId:$tranferTypeId,
    value: $value
    ){
        transactionStatus,
        transactionDate,
        transactionExternalId
    }
}

2. Para obtener una transaccion ejecutar el query:

query($transactionExternalId:String!){
    getTransactionByExternalId(transactionExternalId:$transactionExternalId){
        transactionExternalId,
        transactionType{
            name
        },
        transactionStatus{
            name
        },
        value,
        createdAt
    }
}
